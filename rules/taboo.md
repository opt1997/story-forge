# 禁忌与风控规则

## 硬限制

- 不接入番茄小说。
- 不自动发布。
- 不绕过任何网站风控。
- 不生成用于规避审核、刷量、引流、伪装身份或自动化发布的内容。
- 不保存、展示或请求真实 API key、token、cookie、账号密码、私钥。

## 敏感主题处理

- 避免详细描写现实违法操作、诈骗方法、黑产流程、绕过平台审核方法。
- 避免美化自残、自杀、极端暴力、虐待、仇恨、性剥削。
- 未成年人相关内容必须保持安全边界，禁止性化描写和剥削性情节。
- 涉及疾病、法律、投资等高风险主题时，不提供专业建议伪装成事实指引。

## 平台风控关键词倾向

- 避免以规避检测为目的的变体字、谐音字、拆字和暗号。
- 避免诱导关注、私信、转账、加群、站外交易等平台风险表达。
- 避免把风控机制当成对抗目标。

## 避免的桥段

- “一切都是梦”作为主要结局。
- “突然出现万能贵人”解决所有矛盾。
- 主角没有行动，仅靠别人解释真相。
- 反派无动机作恶。
- 为制造爽点强行降智配角。
- 过度依赖误会且不让人物合理沟通。

## 题材选择建议

- 可以写悬疑、都市、奇幻、轻科幻、亲情、职场、成长、反转故事。
- 冲突应来自人物选择、信息差、价值观冲突或环境压力。
- 即使是爽文走向，也要让主角付出行动和代价。
